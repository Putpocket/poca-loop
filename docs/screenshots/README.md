# Screenshot Guide

GitHub README용 스크린샷은 이 디렉터리에 둡니다. 저작권 있는 실제 포토카드 이미지는 캡처하거나 저장하지 말고, 텍스트 메타데이터 중심 화면만 사용합니다.

## Current Placeholders

- `login.png`: 로그인 화면
- `dashboard.png`: 데스크톱 대시보드
- `mobile-dashboard.png`: 모바일 대시보드

README의 placeholder 경로는 위 파일명과 맞춰져 있습니다.

## Recommended Captures

- `haves.png`: Have 목록과 정식 포카 카드
- `wants.png`: Want 목록과 정식 포카 카드
- `direct-matches.png`: 1:1 매칭 결과와 교환 제안 복사 버튼
- `three-way-matches.png`: 3자 순환 매칭 결과
- `pending-photocard-form.png`: 텍스트 기반 임시 포카 등록 폼
- `admin-pending-photocards.png`: 관리자 임시 포카 검토 목록
- `admin-pending-approval.png`: 새 정식 포카 승인 폼
- `admin-pending-merge.png`: 기존 정식 포카 병합 폼
- `checklist-svg.png`: 공유용 체크리스트 SVG 결과

## Capture Rules

- 실제 아이돌/포토카드 원본 이미지를 포함하지 않습니다.
- 이메일, 토큰, 내부 ID가 과하게 노출된 화면은 피합니다.
- 데모 계정과 샘플 메타데이터만 사용합니다.
- 데스크톱은 1440px 너비, 모바일은 390px 너비 기준을 권장합니다.
